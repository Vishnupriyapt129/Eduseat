# EduSeat – Exam Seating System

## 📌 Description
EduSeat is a Java Swing-based application for managing and generating exam seating arrangements.

## 🚀 Features
- Student login system
- Admin dashboard
- Exam hall allocation
- Seating arrangement generation

## 🛠️ Technologies Used
- Java (Swing)
- MySQL (if used)
- VS Code

## ▶️ How to Run
1. Open project in VS Code
2. Compile:
   javac *.java
3. Run:
   java Main

## 📁 Project Structure
- LoginScreen.java
- StudentLogin.java
- AdminDashboard.java
- SeatingGenerator.java
